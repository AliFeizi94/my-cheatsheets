## Jenkins

## On Mac

### Basic

#### Stop Jenkins

```bash
sudo launchctl unload /Library/LaunchDaemons/org.jenkins-ci.plist
```

#### Start Jenkins

```bash
sudo launchctl load /Library/LaunchDaemons/org.jenkins-ci.plist
```
